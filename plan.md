

# 소성로 히터 센서 이상점 자동 분석 및 엑셀 리포트 생성 상세 계획

---

## 1. 목표 및 요구사항
- 센서별(개소)·시간별 데이터에서 이상치 탐지 및 통계
- 이상치 개수, 기준값 초과 정도, 이상치 비율 등 한눈에 파악
- 엑셀 파일로 자동 리포트(요약·상세·시각화) 생성

---

## 2. 데이터 분석 및 이상치 탐지 전략
### 2.1 데이터 로딩/전처리
- pandas로 CSV(세미콜론 구분) 로드, 결측치/타입 변환, 시간 파싱
- 센서별 컬럼 자동 인식 및 동적 처리

### 2.2 이상치 기준 및 탐지 방법
- 기준값: 평균±3*표준편차(기본), 사용자 지정 임계값 옵션
- Z-score, IQR, Isolation Forest 등 복수 알고리즘 지원
- 급격한 변화(시계열 차분), 결측치도 이상치로 판정

### 2.3 통계 및 요약
- 센서별 이상치 개수, 전체 대비 비율, 최대/평균 초과값
- 이상치 발생 시점·값·초과 정도 상세 기록
- 정상/이상 구간 시각화용 데이터 생성

---

## 3. 엑셀 리포트 자동화 설계
### 3.1 시트 구성
- 시트1: 센서별 이상치 통계 요약(센서명, 개수, 비율, 최대/평균 초과값)
- 시트2: 이상치 상세 내역(시간, 센서명, 값, 기준값, 초과 정도)
- 시트3: 정상/이상 구간 시각화(조건부 서식, 그래프)

### 3.2 조건부 서식/시각화
- 이상치 셀 빨간색, 정상 구간은 기본색
- 센서별 이상치 트렌드 그래프(라인/히트맵)

### 3.3 사용자 편의 기능
- 기준값/임계값 직접 입력 옵션
- 리포트 파일명 자동화(날짜, 분석 조건 포함)
- 분석 결과 로그/요약 텍스트 파일 병행 생성
- CSV 파일 선택/업로드 기능(스크립트/GUI)

---

## 4. 확장 및 실무 적용 고려사항
- 대용량 데이터 처리 속도 최적화(병렬 처리, 샘플링 등)
- 엑셀 조건부 서식·그래프 자동화
- 고장 이력/머신러닝 모델 연계 확장(지도/비지도 학습)
- 관리자 리포트·알림 자동화

---

## 5. 구현 예시 워크플로우
1. 사용자: CSV 파일 선택 → 기준값/알고리즘 옵션 지정
2. 스크립트: 데이터 로딩/전처리 → 이상치 탐지/통계 → 엑셀 리포트 생성
3. 결과: 센서별 이상치 요약·상세·시각화 리포트 자동 생성
